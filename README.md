# 提交 PR 流程：

1. 先 fork 一份代码到自己的名下；
2. 在本地开发测试完成后，将 commit push 到 remote repo；
3. 在页面点击 "new pull request" 提交 PR;
4. 等待 review 完成后即可合并。
